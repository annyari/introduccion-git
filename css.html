Cascading Style Sheets

Qué es CSS?

CSS (Cascading Style Sheets, u Hojas de Estilo en Cascada es una tecnología que se desarrolló con el objetivo de separar la presentación de la estructura.

Es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.

El World Wide Web Consortium (W3C) es el encargado de formular la especificación de las hojas de estilo que servirán de estándar para los agentes de usuario o navegadores.

Uso de CSS

  1. Se utiliza para proporcionar informació sobre el estilo, color, fuente, alineación del texto, tamaño y otras caractrísticas no visuales.
  2. Se ha permitido a través del uso de CSS la eliminación del uso de las tablas para el diseño, siendo estas utilizadas solamente para la muestra de datos tabulados.

Formas de uso de CSS

  1. Mediante un CSS introducido por el autor de HTML, por medio de un estilo en linea, una hoja de estilo interna o una hoja de estilo externa.
  2. Estilos CSS introducidos por las personas que ven el documento de tal manera que se sobreescriben los estilos definidos por el autor.
  3. Los estilos marcados "por defecto" por los user agent para diferentes elementos de un documento HTML, como por ejemplo los enlaces.

Ventajas de CSS?

Algunas ventajas de utilizar CSS (u otro lenguaje de estilo) son:

  1. Control centralizado de la presentación de un sitio web completo con lo que se agiliza de forma considerable la actualización del mismo.
  2. Optimización del ancho de banda de la conexión, pues pueden definirse los mismos estilos para muchos elementos con un sólo selector; o porque un mismo archivo CSS puede servir para una multitud de documentos.
  3. Mejora en la accesibilidad del documento, pues con el uso del CSS se evitan antiguas prácticas necesarias para el control del diseño (como las tablas), y que iban en perjuicio de ciertos usos de los documentos, por parte de navegadores orientados a personas con algunas limitaciones sensoriales.
